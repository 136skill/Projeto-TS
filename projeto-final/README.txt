
Sistema de ficheiros com controlo de acesso. Para adicionarum utilizador ao sistema adicionar ao 
ficheiro autenticação.txt um utilizador com a seguinte semantica
nome,password,email. Para adicionar/remover/alterar utilizadores
  			  alterar o ficheiro "autenticacao.txt".

A senha de abertura será enviada para o email associado ao utilizador que tenta abrir o ficheiro.

-------------------------------------------------------------------------------------------------
|1. executar dependencias.sh									|
-------------------------------------------------------------------------------------------------
|2. executar setup.sh										|
-------------------------------------------------------------------------------------------------
|3. executar make.sh										|
-------------------------------------------------------------------------------------------------
|												|
|
| Aceder aos ficheiros:										|
-------------------------------------------------------------------------------------------------

Ficheiro.txt: cat fuse-3.2.1/build/example/passthrough@exe/TS/ficheiro.txt			|

Autenticacao.txt(apenas root): cat fuse-3.2.1/build/example/passthrough@exe/TS/autenticacao.txt	|
